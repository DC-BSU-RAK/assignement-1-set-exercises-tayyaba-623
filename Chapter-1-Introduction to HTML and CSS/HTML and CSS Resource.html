HTML tags

HTML headings

In HTML, heading tags are used to represent the titles and subheadings of different parts of a webpage. There are six headings in HTML, from <h1> to <h6>. <h1> represents the most important headings while, <h6> represents the less important headings.

Code example

        <h1> It's hot outside </h2>
        <h2> It's hot outside >/h2>
        <h3> It's hot outside </h3>
        <h4> It's hot outside </h4>
        <h5> It's hot outside </h5>
        <h6> It's hot ouside  </h6>

Unordered list

A list that does not follow a numerical order is called an unordered. It is defined by the <ul> tag. Each item in the unordered list starts with the <li> tag. Items in an unordered list are displayed with bullet points. In unordered list, the style of the bullet points can be changed to circle, square, or none using list-style-type property in CSS.

Code example

        <ul>
            <li> Pizza </li>
            <li> Burger </li>
            <li> Cold drink </li>
        </ul>

Circle

        <ul style="list-style-type: circle;">
            <li> Tea </li>
            <li> Ice-cream </li>
            <li> Cake </li>  
        </ul>

Square

        <ul style="list-style-type: square;">
            <li> Noodles </li>
            <li> Cake </li>
        </ul>

None

        <ul style="list-style-type: none;">
            <li> Egg </li>
            <li> Milk </li>
        </ul>

ordered list

A list that follow a specific order is called ordered list. It is represented by <ol> tag. Each item in the ordered list starts with the <li> tag. By default, the items in an ordered list are represented by numbers. However, they can be changed to roman numerals(uppercase, lowercase) or letters(uppercase, lowercase).

Code example

        <ol>
           <li>Coffee<li>
           <li>Chocolate<li>
        </ol>

Uppercase Roman

        <ol type="I">
           <li>Laptop</li>
           <li>Smartphone</li>
        </ol>

Lowercase Roman

        <ol type="i">
            <li>Laptop</li>
            <li>Smartphone</li>
        </ol>

Uppercase letters

        <ol type="A">
            <li>Spaghetti</li>
            <li>Biryani</li>
        </ol>

Lowercase letters

        <ol type="a">
            <li>Spaghetti</li>
            <li>Biryani</li>
        </ol>

<img> tag

This tag adds an image to a webpage. It is known as a self-closing tag because it does not have a closing tag. The height and width attributes are used to adjust the size of the image.

Code example

        <img src="img_cake.jpg" alt="Chocolate cake" width="500" height="400">

<br> tag

<br> tag is use to break a line without starting a new paragraph. It is known as a self-closing tag because it does not have a closing tag.

Code example

Hi everyone <br> I am tayyaba <br> I like spicy food.

<hr> tag

The <hr> tag creates a horizontal line used to divide sections on a webpage. This tag is known as a self-closing tag. The length of the horizontal line can be adjusted using the width attribute.

Code example

        <hr style="width:50px">

The <a> tag is used to create hyperlinks that connect one webpage to another or navigate to specific section within the same page.

CSS property

Background-color

The background-color property is used to set the background color of an element on a webpage. This property is usually applied within the <style> tag. By using the body tag in CSS, the background color of the entire webpage or website can be changed.

Code example

        body {
            background-color: black;
        }

To change the background color of a specific element, the code will be written like this in CSS:

     h2 {
        background-color: pink;
     }

     p {
        background-color: white;
     }

Opacity

This CSS property controls the transparency of an element. The value of opacity starts from 0.0 to 1.0. The higher the value, the less transparent. The lower the value, the more transparent.

Code example

     p {
        background-color: blue;
        opacity: 0.1;
        }
Important point:The value of opacity can effect the transparency of the text as well, making it difficult to read. This issue can be resolved by using RGBA color. It only effects the transparency of the background not the text. RGBA color contains four values the first three represents the color, while the last value represents the opacity value for the background of the element.

Code example

     p {
        background-color: rgba(0, 6, 3, 0.2)
        }

border-radius

The border-radius property is used to round the corners of elements like images, boxes, etc. Each of the four corners can have a different radius value. Bellow the code will use to equally round the corners of the elements.

Code example

        .image {
            border-radius: 14px;
        }

Bellow the code will use to add different border-radius for each four corners.

Code example

        img {
            border-radius: 5px 7px 2px 3px;
        }

Padding

In CSS, padding adds space inside an element. If there is a border, the space inside the border and around the content is called padding.

Code example

        p {
            padding: 20px;
        }

Margin

In CSS, the space outside an element and it's border is called margin. Margin allow to add space on all four sides of an element: right, left, top and bottom.

Code example

        p {
            margin-right: 50px;
            margin-left: 60px;
            margin-top: 100px;
            margin-right: 20px;
        }

font-size

This property of CSS is used to manage the size of the font.

Code example

        h3 {
            font-size: 60px;
        }

font-weight

The font-weight property is used to manage the thickness or lightness of the text. This CSS property allows for adjusting how bold or unbold the text appears.

Code example

       .para {
          font-weight: bold;
       }
    
       .para {
           font-weight: 400;
        }

text-decoration-line

This CSS property allows the addition of decorative lines to text, such as overline, underline, or line-through.

Code example

        p {
            text-decoration-line: underline;
        }

        p {
            text-decoeation: overline line-through;
        }
